# Pizza-Sale----SQL
SQL-based Pizza Sales Analysis project to explore revenue, top-selling items, and customer ordering patterns for business insights.
This project analyzes pizza sales data using SQL to generate key business insights such as total orders, revenue, and customer purchasing patterns. It covers basic to advanced analysis, including identifying top-selling pizzas, most common sizes, revenue contribution, and order trends by time and category. The project also explores cumulative revenue and highlights high-performing products to support data-driven decision-making.
